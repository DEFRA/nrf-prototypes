File requirements:

  • Maximum file size: 10MB
  • Supported formats: .shp, .kml, .geojson, .zip

Supported file formats

  • Shape files (.shp) - ESRI shape file format
  • KML files (.kml) - Google Earth format
  • GeoJSON (.geojson) - Web mapping format
  • ZIP files (.zip) - Containing shape files

File preparation

  • Ensure your boundary is a single polygon
  • Check coordinate system compatibility
  • Remove any unnecessary layers
  • Test the file in a GIS application
